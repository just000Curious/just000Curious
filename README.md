## About

I'm a 2026 AI & Data Science graduate from GCOE Ratnagiri. I build ML systems end-to-end — data pipelines, model training, API deployment, containerization. My focus has always been on getting models into production, not optimizing metrics in a notebook.

During my final semester, I ran two internships simultaneously: built a **no-code SQL generation engine** for Konkan Railway Corporation that replaced manual SQL writing for entire departments, and developed **RAG pipelines and agentic AI systems** at Innomatics Research Labs. Both went into production.
